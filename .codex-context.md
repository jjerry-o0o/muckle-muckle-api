# Project Context

## 1. Project Overview
- Personal finance (household account book) service
- Backend API server for the existing frontend
- Spring Boot MVC structure
- Swagger will be used for API docs

## 2. Tech Stack
- Java 17
- Spring Boot
- JPA (planned)
- H2 for early development (while schema/data is unstable)
- MySQL later via Docker container (persist data locally)
- IntelliJ IDEA

## 3. Coding Preferences
- Prefer concise answers
- Prefer actionable guidance and direct, readable code
- Keep explanations minimal but clear
- Do not modify project source unless explicitly requested

## 4. DB Notes
- H2 first, migrate to MySQL later
- Plan for portability between H2 and MySQL
